# Materia
Organize scattered design assets into a truly usable desktop asset library.

You probably know better (or more clueless) than anyone else about where the design assets are scattered. Screenshots are on the desktop, color swatches are in the notes, vector files are in some "final_final" folder, and font specifications are in a certain Slack message. Materia is a macOS desktop tool that gathers all these items in one place.

## What It Does

- Organize assets with a clear hierarchy: `Workspace > Kit > Group > Asset`
- Import common design files such as `SVG`, `PDF`, `PNG`, `JPG/JPEG`, and `GIF`
- Create native assets for:
  - colors
  - text styles
  - notes
- Browse assets in a three-pane desktop layout with sidebar, browser, and inspector
- Search by name, tags, notes, and color values
- Preview metadata and copy code snippets for design tokens
- Sync across your Apple devices
- Export individual assets or package the whole library into a single `.materiadb` archive
